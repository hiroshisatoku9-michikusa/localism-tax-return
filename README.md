# ローカリズム確定申告書（プロトタイプ v4）

会計データ（CSV）を読み込み、「自分の経済活動がどれだけ地域をめぐったか」を可視化する
**地域経済循環レポート** を生成するアプリ。確定申告書風のセルフレポートを出力します。
**税務署に提出する書類をつくるものではありません。**

## 使い方

`index.html` をブラウザで開くだけ（フォント・ライブラリのCDN読み込みにネット接続が必要）。
会計データの内容がサーバーへ送信されることはなく、すべてブラウザ内で処理されます。

1. **よみこむ** — freee形式CSV（Shift-JIS/UTF-8自動判別）をドロップ。年度・地域・屋号・保存設定もここで。地域キーワード（自動判定用）は編集可
2. **毎月の取引** — 継続取引を自動検出（取引先名がなくても「同一科目×金額±12%×3か月以上」で拾う）。グループ単位で地域内/一部地域内/地域外を一括判定。年間2.4万円以上・上位20グループのみ提示
3. **取引先で見る** — 取引先名・品目の文字列から、辞書（全国・海外サービス／全国チェーン／地域キーワード）で候補を自動付与。信頼度と理由つき。「自信のある候補をまとめて適用」可。名前のないデータでは自動的にスキップ案内
4. **大きな取引** — まだ判定されていない取引のうち年間影響の大きいもの（3万円以上 or 各側合計の3%以上、上位12件）を個別確認
5. **のこりをざっくり** — 未判定分を科目ごとにスライダーで推定
6. **できあがり** — レポート生成。ものさし（残留率α・収入減算率β・一部地域内の残留率・減価償却の扱い）はここで調整

## 指標の定義

- **地域内循環率** = 地域内への支出 ÷ 現金支出合計
- **外貨獲得額** = 地域外からの収入（100%計上）
- **域外流出額** = 地域外へ出ていった支出
- **地域還流バランス** = 外貨獲得 + 地域内支出×残留率α − 域外流出
- 「一部地域内」（地域内店舗の全国チェーン等）は設定した割合（既定30%）だけ地域内あつかい
- 思想的ベースは LM3（Local Multiplier 3, New Economics Foundation）の簡易版

このアプリは採点装置ではなく読解装置です。レポートにもその旨を明記しています。

## 判定の優先順位

個別確認（大口） > 継続取引グループの回答 > 取引先名の回答 > 科目別スライダー推定

## プライバシー設計

- サーバなし・送信なし。明細・回答は localStorage のみ（オフにする設定と全削除ボタンあり）
- フォント・ライブラリは外部CDNから読み込む（会計データとは無関係の通信）。UI上にも明記
- 公開リポジトリ・配布物に**実在の会計データを絶対に含めない**

## 開発メモ

- 1ファイル構成（index.html）、vanilla JS、ビルドなし。依存はGoogle Fonts（Zen Kaku Gothic New）のみ
- プレビュー: `.claude/launch.json` の `localism-app`（http-server, port 8741）
- テストフック: `window.__loadCsvText(text)`
- CSVは「収支区分・勘定科目・金額」列が必須。取引先列が空でも動く（継続検出が金額パターンで補う）
- 辞書は `DICT_EXT` / `DICT_PARTIAL` / `DICT_EXT_HINT` / `REGION_PRESETS`（index.html内）

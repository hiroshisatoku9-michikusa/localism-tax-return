# ローカリズム確定申告書（プロトタイプ v2）

確定申告書と収支データを読み込み、「自分の経済活動がどれだけ地域（石垣島）をめぐったか」を
可視化する **ローカリズム確定申告書** を生成するアプリ。

## 使い方

`index.html` をブラウザで開くだけ（CDNライブラリとWebフォントの読み込みにネット接続が必要）。
PDF・CSVの処理はすべてブラウザ内で完結し、**データはどこにも送信されない**。

1. **申告書をよみこむ** — e-Tax出力PDFをドロップ。帳票種類を自動判別し、個人情報欄（税務署名・住所・氏名・マイナンバー・口座・家族/従業員氏名・支払者名・大家の住所氏名等）を自動黒塗り。手動の塗り足し、ゾーンON/OFF、画像化した黒塗り済みPDFの書き出しに対応。
2. **収支データを入れる** — freee形式CSV（Shift-JIS/UTF-8自動判別）をドロップすると、勘定科目→アプリ科目へ自動マッピングして金額が埋まる。未対応科目は動的に行が追加される。事業主貸/借は自動除外。手入力も可。
3. **お金の行き先** — 科目ごとの「インタビュー」形式。金額の大きい取引（科目合計の15%以上 or 3万円以上、上位6件）は個別に「島内/島外」ボタンで振り分け、残りはスライダーでざっくり。
4. **ものさしを決める** — 島内支出の残留率（既定90%）、島内収入の減算率（既定90%）、減価償却費の扱い、地域名。
5. **できあがり** — ローカリズム確定申告書（HTML）。島内循環率・外貨獲得額・地域経済貢献額の3スコア＋科目別内訳＋循環ゲージ。HTML保存・印刷対応。

## 指標の定義

- **島内循環率** = 島内への支出 ÷ 現金支出合計
- **外貨獲得額** = 島外からの収入（100%計上）
- **調整後流出** = 島外への支出 + 島内への支出 × (1 − 残留率)
- **地域経済貢献額** = 外貨獲得 + 島内支出×残留率 − 島外流出
- 思想的ベースは LM3（Local Multiplier 3, New Economics Foundation）の簡易版

## プライバシー設計

- サーバなし・送信なし。入力値は localStorage のみ（PDFは保存しない。CSV由来の明細は localStorage に保存）
- 自動黒塗りゾーンは `index.html` の `REDACT_PRESETS`（ページ比率座標）で定義。令和7年分・作成コーナー出力のレイアウトに合わせて調整済み

## 開発メモ

- 依存: pdf.js 3.11 / pdf-lib 1.17 / Zen Maru Gothic（いずれもCDN）
- プレビュー: `.claude/launch.json` の `localism-app`（http-server, port 8741）
- テストフック: `window.__loadPdfBuffer(arrayBuffer)` / `window.__loadCsvText(text)`
- CSVは「収支区分・勘定科目・金額」列が必須。取引先列が空でも動く設計（実データで全行空を確認済み）
